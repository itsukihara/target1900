# target1900_webapp

ターゲット1900（Excel）の内容を **サーバ側に内蔵** した、URL共有できる Webアプリです。

## 機能
- 範囲（番号）指定 + 出題数指定 + ランダム
- モード3つ
  1. **PDFテスト**（ブラウザの印刷機能でPDF保存）
  2. **練習**（日本語→英語タイピング / 英語→日本語6択）
  3. **ゲーム**（寿司打風：制限時間120秒固定、正解で+1秒、スコア加算）
- **間違えたら正しく入力/正しい選択をするまで次に進めない**
- ゲームのハイスコアは **礫隊** 1つだけサーバ保存（過去最高のみ更新）

## 起動（ローカル）
```bash
cd target1900_webapp
pip install -r requirements.txt
python app.py
```
http://127.0.0.1:5000

## デプロイ
Render / Fly.io などで Python Web Service としてデプロイしてください。
永続ストレージがない環境だとSQLite（ハイスコア）がリセットされる場合があります。
